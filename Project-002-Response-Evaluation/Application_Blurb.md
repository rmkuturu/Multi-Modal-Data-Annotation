# Application Blurb

I completed a customer-support response evaluation project focused on pairwise judgment of AI-generated answers. The work involved comparing Response A vs. Response B across 100 support tickets, scoring each response for accuracy and helpfulness, identifying hallucinated content, and documenting the rationale behind each preferred-response decision.

To make the project portfolio-ready, I created annotation guidelines, a QA checklist, a project summary, and a reflection document. This project helped me practice an important evaluation skill: distinguishing responses that sound fluent or helpful from responses that are actually grounded, reliable, and useful to the user.

Key skills demonstrated:
- pairwise response evaluation
- hallucination detection
- rubric-based scoring
- QA documentation
- customer-support annotation reasoning
